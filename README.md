## Conky Config 📋💻

Этот конфигурационный файл позволяет создать стильный и функциональный виджет мониторинга системы на рабочем столе Linux.

### Зависимости:

```ini
curl для передачи данных
jq для обработки json
```

### Основные возможности:

- Большие часы с указанием текущего времени и даты, отображаемые в центре экрана.
- Подробная статистика использования ресурсов ПК:
  - Нагрузка процессора (CPU).
  - Загруженность оперативной памяти (RAM).
  - Температура CPU.
  - Информацию о беспроводном соединении Wi-Fi и IP-адресе.
- Полезные блоки для мониторинга состояния жестких дисков:
  - Доступное пространство на каждом разделе диска.
  - Тип файловой системы каждого раздела.
- Актуальная погода для выбранного города.
- Текущие курсы основных валют (доллар, евро).
  (Курс валют парсится с сайта ЦБ РФ, периодически есть проблема с отдачей данных, смена источника не помогла, оставил как есть)
- Список пяти наиболее ресурсоемких процессов, сортируемых по нагрузке на процессор и память.
- Обновляемый блок с заголовками новостей ИТ-тематики.
  (Заголовки фильтруются для показа по ширине окна conky, поэтому обновляются не часто)

### Особенности конфигурации:

- Возможность автоматического обновления показаний.
- Оформление с настраиваемыми цветами и шрифтами.
- Совместимость с различными дистрибутивами Linux.
- Минималистичный дизайн.

---

#### Установка и использование:

Скопируйте этот конфиг в домашний каталог:

```bash
`/home/$USER/.config/conky/conky.conf`
```

Затем запустите Conky следующим образом:

```bash
conky &
```

При желании включите автоматический запуск конфига при каждой загрузке системы, следуя инструкциям ниже.

---

### ⚙️ Настройка Автозагрузки

Создайте файл autostart:

```bash
mkdir -p ~/.config/autostart && nano ~/.config/autostart/conky.desktop
```

Добавьте следующую запись:

```ini
[Desktop Entry]
Type=Application
Name=Conky
Exec=/usr/bin/conky
Comment=Запуск Conky
NoDisplay=false
X-GNOME-Autostart-enabled=true
Hidden=false
Terminal=false
```

Сохраните изменения и перезагрузитесь для проверки.

---

### Файлы

- `/home/$USER/.config/conky/conky.conf`: основной конфигурационный файл.
- `~/.config/autostart/conky.desktop`: настройка автозапуска.

### 🛠️ Автоматизация установки и настройки Conky

Данный скрипт предназначен для быстрой установки и настройки утилиты **Conky**, популярной панели мониторинга ресурсов компьютера в среде Linux. Конфиги будут настроены таким образом, чтобы отображалась полезная информация о состоянии вашей системы (например, нагрузка процессора, температура, сетевые подключения и другое). Ниже подробно описаны ключевые моменты работы скрипта.

---

### ⚙️ Основные этапы работы скрипта:

1. **Подготовка окружения**
   - Определяется текущий пользователь и домашняя директория.
   - Создаются необходимые директории (`.config/conky`) и подгружается конфигурационный файл `conky.conf` с удалённого сервера.

2. **Автозапуск Conky**
   - Генерируется специальный файл автозапуска (`autostart`), благодаря чему Conky стартует вместе с системой при каждом входе пользователя.

3. **Проверка наличия приложения Conky**
   - Если приложение не установлено, выполняется проверка операционной системы. Если ОС основана на Arch Linux, устанавливается нужное ПО автоматически. Иначе выдается рекомендация вручную установить программу.

4. **Запуск Conky**
   - По завершении всей процедуры автоматически запускается установленный экземпляр Conky.

---

### ✅ Пользовательские преимущества:

- Полностью автоматизированная процедура установки и настройки.
- Возможность быстрого развертывания рабочего окружающего мониторинга.
- Поддержка удобной автозагрузки при старте системы.
- Оптимизированный конфиг, обеспечивающий понятное и информативное представление данных.

---

### 📌 Примечания:

- Этот скрипт ориентирован преимущественно на дистрибутивы Linux семейства Arch Linux. Если вы используете другой дистрибутив, вам может потребоваться дополнительная адаптация некоторых шагов.
- Обязательно удостоверьтесь, что файл `conky.conf` доступен по указанному адресу.
